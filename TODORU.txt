Особенности

	•	Установка сервера с выбором IP-адреса, порта машины и т.д.
	•	Отображение имени пользователя перед его сообщением.
	•	Имя пользователя запрашивается только один раз при входе в комнату.
	•	Команда /list должна показывать людей в комнате только тому, кто запустил команду.
	•	Сервер должен быть неуязвимым.
	•	Рассмотреть возможность настройки.
	•	Управление кодами ошибок в сервере и клиенте.
	•	Реализация владельца (первого, кто создал комнату).
	•	Изменение пароля текущей комнаты с помощью /hodor <newpass>.
	•	Управление чат-комнатой с помощью команд вроде /ban <username>.
	•	Предоставление статуса владельца с помощью команды /powerto <username>.
	•	Очистка текущей комнаты с помощью команды /erased <roomname>.
	•	Перенаправление участников из одной комнаты в другую с помощью команды /axios <CurrentRoom> <NewRoom>.
	•	Команда для аварийного выключения сервера.
